The CoinMarketCap API has evolved continuously since its launch, expanding from a simple listings endpoint into a comprehensive financial data platform covering spot markets, DEX on-chain data, WebSocket streams, and AI-native integrations. This page summarizes what changed in each major version and highlights recent additions — so you can plan upgrades and take advantage of every new market intelligence capability as it ships.

<Note>
  All previous API versions remain fully available for backward compatibility. Existing integrations built on v1, v2, or v3 continue to work without modification. Upgrading to a newer version is opt-in and recommended only when you want to access new capabilities.
</Note>

## API v4 (Current)

API v4 is the current recommended version and the foundation for all new endpoint development. It represents CoinMarketCap's most significant platform expansion to date — bringing on-chain intelligence, AI-native tooling, and frictionless access options that open entirely new categories of market analysis.

**What's new in v4:**

- **DEX endpoints** — Query decentralized exchange pairs, liquidity pools, on-chain token data, and DEX holder analytics across major chains. Discover emerging opportunities before they reach centralized exchanges.
- **On-chain data** — Access token contract metadata, wallet holder distributions, and chain-specific market metrics for deeper due diligence on any asset.
- **x402 pay-per-request** — Call the API with no account or API key using the x402 protocol; micro-payments of $0.01 USDC are deducted per request, making market data accessible with zero friction.
- **WebSocket streams** — Subscribe to real-time price and volume updates pushed directly to your client over a persistent connection — the foundation for responsive trading dashboards and live alerting.
- **MCP server** — AI-native market intelligence integration at `mcp.coinmarketcap.com/mcp` for use with Cursor, Claude Code, Windsurf, and other MCP-compatible AI tools. Ask your AI assistant live market questions and get real-time answers from the world's most-referenced crypto data source.

## API v3

API v3 focused on depth and precision for data-intensive applications, giving developers the granularity needed to build sophisticated research and analytical tools.

**Key additions in v3:**

- Enhanced historical data endpoints with finer granularity and longer lookback windows, enabling richer backtesting and trend analysis.
- Expanded quote endpoints supporting bulk symbol lookups in a single request — dramatically reducing the calls needed to monitor a diversified portfolio.
- Improved pagination using `start` and `limit` parameters across listing endpoints for efficient large-scale data retrieval.
- Additional metadata fields on cryptocurrency objects, including tag arrays and platform contract addresses, supporting deeper asset screening workflows.

## API v2

API v2 introduced multi-asset support and expanded the exchange data offering, laying the groundwork for portfolio-level market monitoring.

**Key additions in v2:**

- `/v2/cryptocurrency/quotes/latest` — Multi-coin quote lookups returning data for many assets in one call, replacing the single-asset v1 endpoint and making portfolio tracking far more efficient.
- Exchange data improvements — listing, metadata, and market pair endpoints for centralized exchanges, giving developers full visibility into where volume and liquidity flow.
- Expanded `convert` parameter support for fiat and crypto conversion across quote endpoints.
- Introduction of the CMC ID system as the canonical, stable identifier for all assets and exchanges — the foundation for reliable, long-running data integrations.

## API v1 (Original)

API v1 established the core data model and the endpoints that made CoinMarketCap the reference data source for the entire crypto industry.

**Original v1 endpoints:**

- `/v1/cryptocurrency/listings/latest` — Paginated ranked list of all cryptocurrencies by market cap.
- `/v1/global-metrics/quotes/latest` — Total market cap, BTC dominance, and active cryptocurrencies count.
- `/v1/exchange/listings/latest` — Ranked exchange listings with volume data.
- `/v1/tools/price-conversion` — Currency conversion utility across fiat and crypto.

## Recent Highlights

The following features shipped as part of ongoing v4 development and are available now — each one expanding the analytical edge available to developers and traders building on CoinMarketCap data:

- **x402 protocol launch** — Pay-per-request access with no account or API key required, powered by the x402 open payment standard. Get started building in seconds at $0.01 USDC per call.
- **MCP server** — AI-native market data integration for IDE-based AI assistants via the Model Context Protocol. Query live prices, rankings, and DEX data directly from Cursor, Claude Code, or Windsurf. See the [MCP Server guide](/tools/mcp-server) to connect in minutes.
- **DEX Holders endpoints** — On-chain holder analysis including wallet concentration, distribution curves, and holder change over time — powerful signals for assessing community strength and conviction behind any token.
- **Altcoin Season Index API** — Programmatic access to the popular Altcoin Season Index, enabling applications to read the current market cycle indicator in real time and build cycle-aware investment strategies.
- **Real World Assets (RWA) tracking** — Dedicated category and metadata support for tokenized real-world assets including treasury bills, real estate, and commodities — capturing one of the fastest-growing segments in crypto markets. See the [RWA listings endpoint](/api-reference/rwa/listings) to start integrating.
