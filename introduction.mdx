## Three Steps to Live Data

<Steps>
  <Step title="Get Your API Key">
    Create a free account at [pro.coinmarketcap.com/signup](https://pro.coinmarketcap.com/signup). Once registered, your API key is available instantly in the dashboard under **API Keys**. The free Basic plan includes 15,000 credits per month — more than enough to start building.
  </Step>
  <Step title="Make Your First Call">
    Pass your key in the `X-CMC_PRO_API_KEY` header and hit your first endpoint. The `/v2/cryptocurrency/quotes/latest` endpoint returns real-time price, volume, market cap, and percent-change data for any token by symbol or ID.

    ```bash
    curl -H "X-CMC_PRO_API_KEY: YOUR_API_KEY" \
      "https://pro-api.coinmarketcap.com/v2/cryptocurrency/quotes/latest?symbol=BTC"
    ```
  </Step>
  <Step title="Explore the Full Data Universe">
    Once the first call succeeds, the entire platform is at your fingertips. Query historical OHLCV data, stream live price ticks via WebSocket, analyse DEX liquidity pools, pull on-chain metrics, or integrate with AI workflows through our MCP server — all from the same base URL: `https://pro-api.coinmarketcap.com`.
  </Step>
</Steps>

## Base URL

All API requests target a single base URL:

```
https://pro-api.coinmarketcap.com
```

Every endpoint is versioned (e.g. `/v1/`, `/v2/`) to ensure backwards compatibility as the platform evolves. The full endpoint reference lives at [coinmarketcap.com/api/documentation/v1/](https://coinmarketcap.com/api/documentation/v1/).
