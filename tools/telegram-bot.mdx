With the CoinMarketCap Telegram bot, you can pull live cryptocurrency prices, check trending tokens, and convert between currencies without ever leaving your chat. Whether you're in a group discussion or a one-on-one conversation, @CoinMarketCapBot delivers market data the moment you need it.

## Getting Started

<Steps>
  <Step title="Find the bot on Telegram">
    Open Telegram on any device and search for **@CoinMarketCapBot** in the search bar. Tap the official CoinMarketCap bot from the results list.
  </Step>
  <Step title="Start the bot">
    Tap **Start** or send the `/start` command. The bot will introduce itself and display a quick-start menu of available commands.
  </Step>
  <Step title="Query your first asset">
    Type any cryptocurrency ticker symbol — for example, `BTC`, `ETH`, or `SOL` — and the bot will return the current price, 24h change, and market cap instantly.
  </Step>
</Steps>

## Commands Reference

| Command | Description |
|---------|-------------|
| `/start` | Initialize the bot and display the welcome message with available commands. |
| `/price [SYMBOL]` | Get the current price and 24-hour percentage change for a cryptocurrency (e.g., `/price BTC`). |
| `/top` | Display the top 10 cryptocurrencies ranked by market capitalization. |
| `/trending` | Show the currently trending tokens based on CoinMarketCap watchlists and page views. |
| `/convert [AMOUNT] [FROM] [TO]` | Convert a value between any two currencies (e.g., `/convert 1.5 ETH USD`). |
| `/help` | List all available bot commands with usage examples. |

## Inline Mode

You don't need to open a private chat with the bot to use it. Inline mode lets you query CoinMarketCap from **any Telegram conversation** — group chats, channels, or direct messages.

To use inline mode:

1. In any chat's message box, type `@CoinMarketCapBot` followed by a space and a ticker symbol (e.g., `@CoinMarketCapBot BTC`).
2. A live price card will appear above the keyboard as an inline result.
3. Tap the card to post it directly into the conversation.

This makes it effortless to share live prices with friends, trading groups, or colleagues — no copy-pasting required. The shared card includes the current price, 24h change, and a direct link to the full CoinMarketCap asset page.

<Warning>
  Inline results are fetched in real time. Make sure your Telegram app has an active internet connection to receive up-to-date data.
</Warning>

<Tip>
  Use the Telegram bot for quick on-the-go price checks and ad-hoc conversions while letting the CoinMarketCap REST API handle your automated workflows, scheduled reports, and application back-end data needs — the two tools complement each other perfectly.
</Tip>
